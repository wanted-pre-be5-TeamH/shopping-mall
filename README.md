## 1. 프로젝트 목적
- 제품 쇼핑몰 관리페이지의 백엔드 개발
- 기능 : 주문 관리 서비스, 쿠폰 관리 서비스

## 2. 백엔드 프로젝트 구성
1) 서버 : Nestjs
2) DB : PostgreSQL (Docker)
3) ORM : Prisma
4) Unit Test : Pactum

## 3. API 명세
